
# Demographic Data Dashboard for India

## Objective
Design an  end-to-end  workflow that ingests, cleans,  analyses, and visualises  demographic data for  India,
culminating in an interactive dashboard that supports exploratory and comparative analysis.
## Core Tasks
## •

of India, NFHS, Sample Registration System, or open government data portals).
•Ingest data frommultiple sources or files, potentially at different geographic resolutions (national,
state, district).

–Inconsistent geographic identifiers
–Missing, suppressed, or censored values
–Temporal mismatches across indicators or survey years
## •
Perform exploratory analysis on key demographic indicators such as population size, literacy rate,
sex ratio, age distribution, fertility, or urbanisation.
•Design and implement aninteractive dashboardthat allows users to:
–Compare demographic indicators across regions
–Explore distributions and regional disparities
–Filter by geography and demographic variables

## Expected Outcome
A  reproducible  demographic  data  pipeline  and  interactive  dashboard  that  demonstrate  rigorous  data
ingestion  and  cleaning,  thoughtful  exploratory  analysis,  clear  visual  communication,  and  responsible
interpretation of population-level data.
