# Datasets

## 1. District Sex Ratio Dataset:

File Name:
District, Sex Ratio_2001_Rural, Sex Ratio_2001_Urban, Sex Ratio_2001_Total, Sex Ratio_2011_Rural.csv

### Number of rows and columns

Rows: 228 (district records)

Columns: Sex ratio values for different census years and areas

### What this dataset contains

This dataset shows the sex ratio of districts in India for Census 2001 and Census 2011. It compares rural, urban and total population sex ratios.

### Columns 

District: Name of the district

Sex Ratio 2001 Rural: Number of females per 1000 males in rural areas in 2001

Sex Ratio 2001 Urban: Number of females per 1000 males in urban areas in 2001

Sex Ratio 2001 Total: Overall sex ratio in 2001

Sex Ratio 2011 Rural: Female to male ratio in rural areas in 2011

Sex Ratio 2011 Urban: Female to male ratio in urban areas in 2011

Sex Ratio 2011 Total: Overall sex ratio in 2011

Decadal Change: Change in sex ratio between 2001 and 2011

### Definition
Sex Ratio: Number of females per 1000 males in the population






## 2. Fertility (%) – DLHS IV Dataset:

File Name:
Fertility (%) - DLHS IV.csv

### Number of rows and columns

Rows: 295

Columns: State, district and fertility indicators

### What this dataset contains

This dataset comes from the District Level Household Survey (DLHS-IV). It provides fertility related statistics for districts in India

### Columns explained

States: Name of the state

Districts: Name of the district

Fertility indicators (%): These columns show fertility related percentages such as: fertility rate, birth related indicators, reproductive statistics for women

### Definition
Fertility Rate: Average number of children born to women during reproductive age (15-49 years)






## 3. Fertility Indicators for Major States (2006–2013):

File Name:
Fertility Indicators for Major States of General Marital Fertility Rate and Total Marital Fertility Rate from 2006 to 2013.csv

### Number of rows and columns

Rows: 168

Columns: State, year and fertility indicators

### What this dataset contains

This dataset shows fertility indicators for major Indian states from 2006 to 2013. It focuses on fertility among married women

### Columns explained

State: Name of the state

Years: Year of data (2006–2013)

General Marital Fertility Rate (GMFR): Number of births per 1000 married women aged 15–49 years

Total Marital Fertility Rate (TMFR): Average number of children a married woman is expected to have during her lifetime






## 4. State wise TFR and MCPR Dataset:

File Name:
State wise TFR and MCPR 2011.csv

### Number of rows and columns

Rows: 37 (States and Union Territories)

### What this dataset contains

This dataset shows fertility and contraceptive use in different states of India in 2011

### Columns explained

States/UT: Name of the state or union territory

TFR (Total Fertility Rate): Average number of children a woman is expected to have during her lifetime

MCPR (%) – Modern Contraceptive Prevalence Rate: Percentage of married women using modern contraceptive methods






## 5. Total Fertility Rate India Dataset:

File Name:
Total Fertility Rate India 2011.csv

### Number of rows and columns

Rows: 21 (states)

Columns: fertility rate values for many years

### What this dataset contains

This dataset shows Total Fertility Rate trends in India from 1971 to 2010 for different states

### Columns explained

State: Name of the state

Year columns (1971–2010): Each column represents the Total Fertility Rate for that year
